### Features

---

### Install

First install [docker](https://docs.docker.com/get-docker/) and [docker-compose](https://docs.docker.com/compose/install/), and then:

```shell
$ git clone https://github.com/mohamad-zahiry/url_shortener.git
$ cd url_shortener
$ docker compose build
$ docker compose up
```
